# TFT Reference Sheets

External Google Sheets holding **Teamfight Tactics (TFT)** set 9, 10, and 11 data — heroes, traits, and skills. We use these to analyze the auto-chess genre and inform the design of this game (an auto-battler; see `Assets/Scripts/Battle/`).

These are the **canonical sources** whenever a design doc or note refers to "the TFT set9 sheet", "set10 sheet", etc.
